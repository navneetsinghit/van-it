Driver-dashboard User Stories

- A driver can click a link to go to the user edit profile page
- A driver can mark themselves as un/available

- If a job is available
	- A driver can see that a job is available
	- A driver can see a map that shows where the job will commence
	- A driver can accept the available job
	- A driver can decline the available job
- If no jobs are available
	- A driver can see an "No jobs available" notification

- A driver can click a link to go to their job history page
